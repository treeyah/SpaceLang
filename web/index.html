    Commands
    write(text)

    Print text to the output.

    write(Hello) → Hello
    add(a+b)

    Print the sum of two whole numbers.

    add(10+5) → 15
    minus(a-b)

    Print the difference.

    minus(10-4) → 6
    times(a*b)

    Print the product.

    times(6*7) → 42
    divide(a/b)

    Print the quotient (whole-number division). Dividing by zero shows an error.

    divide(20/4) → 5
    set(name=value)

    Store a variable for later use.

    set(user=Ben)
    get(name)

    Print the value of a stored variable.

    get(user) → Ben
    repeat(n)

    Run the next command n times.

    repeat(2)
    write(Hi) → Hi (twice)

    Rules: no spaces inside the brackets, and math uses whole numbers only. You don't need to type run() — the Run button does that for you.
