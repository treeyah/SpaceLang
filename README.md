                                  ==SpaceLang==
                                  
This is a project teaching children how to code. This is not the final version and I don't expect anyone to get deep in it.

Codes:
write(): anything in the bracket is written on the screen. It is like print. Example: write(hello)

add(): anything in the bracket is the addition sum. Example: add(5+3)

minus(): same as add() but with subtraction. Example: minus(4-3)

times(): multiplication. Example: times(5*5)

divide(): division. Example: divide(12/2)

set(): sets any variable. Example: set(name=john) or set(x=123)

get(): writes the variable. Example: get(name)

repeat(): repeats anything under it. Example: 
repeat(5)
write(hello)
